# 📈 Zerodha Trading Dashboard Clone

**By Aditya Mahekar**

A modern stock trading dashboard inspired by Zerodha Kite.
This project provides a complete trading interface including portfolio tracking, stock watchlist, order management, and analytics visualization.

---

## 🧠 Features

* 📊 Dashboard Overview – View margin, balance, and account summary
* 📈 Stock Watchlist – Track stocks with price changes
* 💸 Buy/Sell System – Execute trades directly from UI
* 📁 Holdings Management – Track investments with P&L
* 📜 Orders Section – Monitor trade activity
* 📊 Portfolio Analytics – Pie chart & bar graph visualization
* ⚡ Real-Time Feel – Dynamic UI updates
* 🎨 Clean UI – Inspired by Zerodha Kite
